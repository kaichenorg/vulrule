BIO_eof

opensslReturn Value Check

API 概述​

BIO_eof 是 openssl 库中的一个API。该API属于 Return Value Check 类型的规则。

使用规则​

由 AURC 生成

规则 1

Return Value Check

Description

BIO_eof() returns 1 if EOF has been reached, 0 if not, or -1 for failure.

CweType

CWE-253

Label

return value check

QLCode

import cpp

class OpenSSLFunctionCall extends FunctionCall {
  OpenSSLFunctionCall() {
    this.getTarget().hasName("BIO_eof")
  }
}

from OpenSSLFunctionCall call, UnaryOperation uop
where
  uop.getOperator() = "!" and
  uop.getOperand() = call.getAnAccess()
select uop, "This negation checks the return value of BIO_eof."
